# System Call Demonstrator (Python)

## Overview
This project demonstrates operating system system calls and process management concepts using Python. It helps understand how programs interact with the operating system through different system-level operations.

## Features
- Process creation and management
- File handling operations
- System command execution
- OS interaction using Python modules
- Demonstration of basic operating system concepts

## Technologies Used
- Python
- OS Module
- Subprocess Module
